✉️ Quick Add via Email

Anyone in the family can add to the dashboard by sending an email. Forward any email or send a new one to the family HQ address with a simple subject line format.

📬 Step 1: Set up the family email
Create a new Gmail account (e.g. stimson.family.hq@gmail.com) — or use an existing family email.
📝 Step 2: Use these subject line formats
🛒 SHOPPING: Item | Quantity | Category
    Example: SHOPPING: Milk | 2 pints | Dairy

✅ CHORES: Task | Who | Frequency
    Example: CHORES: Empty bins | Simon | Weekly

📅 EVENT: What | Date (DD/MM/YYYY) | Time | Who | Location
    Example: EVENT: Ava dentist | 15/06/2026 | 14:00 | Ava | Hitchin

📝 NEWS: Message | Priority
    Example: NEWS: Trip payment due | High

📞 CONTACT: Name | Phone | Email | Type | Notes
    Example: CONTACT: Dr Smith | 01462 123456 | dr@smith.co.uk | Doctor

🍽️ MEAL: Day | What
    Example: MEAL: Monday | Spaghetti Bolognese
⚙️ Step 3: Set up automatic processing
  1. Go to script.google.com and create a new project
  2. Copy the email_processor.gs script (available in the FamilyHQ repo under scripts/)
  3. Paste it into the script editor
  4. Run testProcessEmails first to check it works
  5. Add a trigger: Edit → Triggers → Add trigger → processEmails → Every 5 minutes
  6. Share the Family HQ spreadsheet with the new Gmail account
💡 Tips for the family
• Use the subject line for the command — the email body is ignored
• SHOPPING items go straight to the shopping list
• EVENTS go to the calendar (dates work in DD/MM/YYYY format)
• NEWS appears in the announcements tab
• The system checks every 5 minutes — so items appear quickly!
• Processed emails get a "FamilyHQ" label in Gmail so you can see what's been added
